# :dev_mysql_original: Data elimination - Delete Sentence

Deletes any row (selected in the condition with `WHERE`) from the table of the database
## Syntax

```sql title="deleteSentence"
DELETE FROM tableName
[WHERE condition]
[ORDER BY customOrder];
```
