# :dev_mysql_original: Eliminación de registros sentencia delete

Permite eliminar cualquier fila de las tablas de la base de datos

# Sintaxis

```SQL title:Sentencia_delete
DELETE FROM nombre_tabla
[WHERE condición]
[ORDER BY columna_ordenación];
```
