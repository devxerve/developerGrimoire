# :dev_java_original:🔒Java - Accesmodifiers
Acces modifiers in Java control attributes’s and method’s visibilty inside a Class.
There are for main levels
`public`: Accesible from every class in every package
`protected`: Accesssible inside the same package and subclasses
`private` : Only accessible within the same class
default ( no modifier ) : Only accessible within the same package
