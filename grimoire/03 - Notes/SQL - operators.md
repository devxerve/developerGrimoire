# Operadores relacionales

```SQL  title="Relacionales SQL"
WHERE edad        >  18
WHERE precio      <  100
WHERE salario     >= 1500
WHERE cantidad    <= 20
WHERE nombre      =  'Juan'
WHERE tipo        != 'admin'
WHERE puntuacion  !> 80
WHERE ranking     !< 5
```

# Operadores logicos
```SQL title="Operadores_lógicos_SQL" group="Operadores Logicos"
WHERE edad > 18 AND ciudad = 'Madrid'
WHERE precio < 100 OR cantidad > 5
WHERE NOT estado = 'activo'
```
