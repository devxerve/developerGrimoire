# :dev_java_original:Datos Primitivos java
Estos datos se pueden convertir a otro tipo de datos con Casting
````java cpp title:Datos_primitivos
byte nombre = 0; //Número entero de 8 bits
short nombre = 0; //Número entero de 16 bits 
int nombre= 0; // Número entero de 32 bits
long nombre = 0; //Número entero de 64 bits
float nombre = 0F; //es un dato de coma flotante, un número decimal de 32 bits
double nombre = 0; //es un dato doble, número decimal de 64 bits, sirve para cálculos mas exactos 
boolean nombre = false; //Valor lógico, True o False (0 Ó 1)
char nombre = \u0000; // Carácter Unicode de 16 bits
````
