# Programación concurrente
Permite la posibilidad de tener en ejecución al mismo tiempo múltiples tareas interactivas. 
Permite generar procesos e Hilos.
# Programación secuencial
No permite la posibilidad de tener en ejecución múltiples tareas, funciona como una serie de pasos escalonados, un paso no se ejecutara si no se ejecuta el anterior, y el siguiente a el no se ejecutara hasta que dicho paso haya terminado. 

# Ejemplo programación secuencial
`'java' retorno = proceso.waitFor();` <mark style="background: #FF5582A6;">IMPORTANTE, SI ESTO ESTA EL PROGRAMA ES SECUENCIAL</mark>
