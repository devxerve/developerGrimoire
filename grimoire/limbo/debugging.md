# :dev_vscode_original: Depuración Del Código
La depuración permite <mark style="background: #FF5582A6;">identificar y corregir errores de programación mediante la ejecución controlada del software</mark>, Esta tarea la realiza el depurador o *Debugger* 

> [!info] Debugger / Depurador
> Gracias al depurador podemos ver el proceso de un programa paso a paso, observando los valores de nuestros metodos, variables y objetos

Hay otro modo que es usando puntos de interrupción para controlar partes del software independientemente.
