# :dev_html5_original: Html5 - Introducción Y Estructura Básica

## Html5
El lenguaje HTML5 se guía por Etiquetas estas etiquetas aportan información o formato al contenido algunas tienen una series de atributos que las modifican, la estructura es < etiqueta atributos > </ etiqueta> **Todas las etiquetas deben tener apertura y cierre para funcionar** *
<sub>*Hay etiquetas como meta que se cierran en la misma línea </sub>
## **Atributos**
Algunas etiquetas tienen atributos que modifican su contenido, **todos los atributos se ponen de esta manera** : < elemento nombreDeAtributo1=”valor1” nombreDeAtributo2=”valor2” ... >          </ elemento>
## **Estructura Básica HTML5**
Para crear una página web usando HTML5 esta es la estructura básica, el navegador elimina las tabulaciones y espacios en blanco que pongamos en el código. (Mirar Etiquetas para entender cada una de las etiquetas mencionadas.)
```html
<!DOCTYPE html>
<html lang="es">
	<head>
		<meta charset="UTF-8">
		<title>Estructura</title>
	</head>
	<body>
		CONTENIDO CON ETIQUETAS
	</body>
</html>
```
