# Apuntes
## Etiquetas De Bloque
## Html
- Sirve para delimitar el inicio y fin de cada fichero HTML
	- Atributos: lang - sirve para poner el lenguaje de la página web
```html
<html lang ="es">
	Esto es un documento HTML
</html>
```
## Head
- Sirve para insertar contenido importante que no se muestra en la página (Titulo, metadatos, estilos, ficheros enlazados....)
```html
<head>
	Aqui irian una serie de datos del documento por ejemplo:
	<meta>
	<title>Titulo</title>
	<link> linkea a otro archivo, fichero... etc </link>
</head>
```
## Meta
Sirve para incrustar metadatos, se cierra en la etiqueta de apertura tal que asi:
```html
<meta (dato bla bla bla) >
```
## Body
Delimita el contenido de la web
```html
<body>
	Esto es la web
</body>
```

## P
Crea un párrafo en el texto
```html
<p> Esto es un párrafo</p>
```
 
<p> Esto es un párrafo</p>

## Etiquetas Línea
## H1
- Sirve para delimitar titulos
	- IMPORTANTE: Solo puede existir un h1 en el fichero html, hay hasta 6 etiquetas h (h1, h2, h3, h4, h5, h6,)
```html
<h1>Esto es un título</h1>
```

<h1>Esto es un título</h1>

## A
- Crea un enlace 
	- atributo href - sirve para poner la página web o archivo enlazado de destino.
```html
<a href="www.google.com"> google</a>
```

<a href="www.google.com"> google</a>


## Img
- muestra una imagen
	- atributo src - sirve para poner la ruta de la imagen a mostrar
```html
<img src="ruta_imagen.jpg"/>
```

<img src="ruta_imagen.jpg"/>
