# Estructura Del Documento Xml

Un documento XML se organiza en dos secciones principales: el **prólogo** y el **ejemplar**
El prólogo es opcional y puede incluir la declaración de la versión, la codificación (como UTF-8) y si el documento es autónomo. El ejemplar es la parte obligatoria y contiene los **datos reales** estructurados mediante elementos anidados
