## Funcionamiento Básico De Un Sistema Informático

Un ordenador funciona mediante un ciclo continuo:

1. **Datos de Entrada**: Se introducen mediante dispositivos como ratón, teclado, escáner, conexión de red o unidades de almacenamiento

2. **Procesamiento**: La CPU, apoyada por RAM, chipset y tarjeta gráfica, ejecuta el programa y procesa los datos

3. **Datos de Salida**: Se obtienen mediante monitor, impresora, pendrive, disco duro, red u otros dispositivos
