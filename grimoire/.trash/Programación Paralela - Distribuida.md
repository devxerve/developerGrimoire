# Programación Paralela / Distribuida
Significa que se programa desde diferentes fuentes, y se coordina entre las fuentes para unir los diferentes módulos
Esto se puede hacer mediante [[Git - Github|Git o cualquier otro sistema de control de versiones]]

# Arquitecturas Distribuidas
Cuando un sistema esta distribuido significa que el sistema esta alojado en varios componentes a la vez.
