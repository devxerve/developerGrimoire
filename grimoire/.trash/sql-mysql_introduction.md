# :dev_mysql_original: Introducción mysql

## Consola de mysql

La conexión a la base de datos se hace mediante este comando
```bash prompt="cmd" title="Conexion windows mysql shell" group="shell"
cd Program Files/MySQL/MySQL Server 8.3/bin
mysql -u root -p
```
```bash prompt="bash" title="Conexión linux mysql shell" group="shell"
mysql -u root -p #mysql en PATH (normalmente)
```
