# :dev_javascript_plain: Programación Web (cliente)
Cubre las necesidades de visualización, interacción y recopilación de datos por parte del usuario, además del envío de dicha información a la parte que actúa como servidor para su posterior tratamiento.
## Components
:dev_html5_original: Lenguaje de marcas HTML
:dev_css3_original: Hojas de estilo CSS
:dev_java_plain:applets Java
:dev_javascript_original: Javascript
:luc_chrome: Navegador web 

Los lenguajes del lado del cliente son aquellos que pueden set interpretados directamente por el navegador
