El papel de los Atributos

Los **atributos** proporcionan propiedades adicionales a los elementos y se declaran siempre dentro de la etiqueta de inicio
A diferencia de los elementos, el **orden de los atributos no es relevante**, pero no se permite repetir el nombre de un atributo dentro del mismo elemento. Se recomienda usarlos para datos de pequeño tamaño que guían el procesamiento pero que no suelen mostrarse directamente al usuario
