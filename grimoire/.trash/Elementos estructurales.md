# Elementos Estructurales
## Div
Se utilizaba como contenedor de otros elementos para facilitar la manipulación de todos a la vez. Dentro se pueden colocar todo tipo de etiquetas. Por defecto no tiene formato, pero se utilize con CSS para dar estilo y crear capas que pueden maquetarse libremente por la página web.
```html
<div style="background-color: beige;float:right">
    <p>Esto es un div combinado con Css para un color de fondo beige y flotante a la derecha</p>
</div>
```

## Span
Es muy similar al anterior pero se usa para marcar contenido en párrafos, diferenciándolo del resto del mismo.
```html
<p>Esto es un parrafo normal <span style="background-color: black;">Aqui el fondo esta negro</span> Aqui no</p>
```
