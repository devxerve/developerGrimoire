# :obs_run_command: Shell De Commandos

## En Linux

El shell es parte fundamental del sistema operativo. Se puede usar con o sin interfaz gráfica. Las capas superiores (gestor de ventanas, escritorio) son modulares.

**Shells más comunes:**

- **bash** (Bourne Again Shell): Shell por defecto, el más utilizado

- **sh** (Bourne Shell): El original

- **zsh** (Z Shell): Más avanzado

- **fish** (Friendly Interactive Shell): Más moderno, autocompletado y colores

- **csh y tcsh** (C Shell): Antiguos, menos usados

- **dash** (Debian Almquist Shell): Usado en Debian


## En Windows

cmd.exe y PowerShell NO forman parte del sistema operativo, son aplicaciones de usuario que usan la API de Windows.

## Diferencia De Arquitectura

|Aspecto|Linux|Windows|
|---|---|---|
|Shell|Parte clave del SO|Aplicación de usuario|
|Interfaz gráfica|Opcional y modular|Obligatoria e integrada|
|Sistema de ventanas|Reemplazable (X11/Wayland)|Integrado y propietario|
|Uso del shell|Potente y estándar|Limitado a tareas administrativas|
