# :luc_codesandbox: Máquinas Virtuales

# Definición
Permiten ejecutar varios sistemas operativos en el mismo ordenador de forma simultánea o alternativa, sin interferencia entre ellos.

## Ventajas

- Múltiples sistemas en una sola máquina física
- Aislamiento entre sistemas
- Fácil instalación y desinstalación de SO
- Pruebas de compatibilidad
- Seguridad (aislar aplicaciones problemáticas)
