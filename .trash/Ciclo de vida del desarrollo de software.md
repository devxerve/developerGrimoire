# Ciclo De Vida Del Desarrollo De Software
# Fases
### Análisis
se definen los requisitos del software que hay que desarrollar
### Diseño
Se determina el funcionamiento de forma general
### Codificación
Se programa el software definido
### Pruebas
Se busca confirmar que el software no tiene errores, y que have lo que tiene que hacer, que no es lo mismo
### Documentación
Se muestra una información completa y de calidad que ilustre como manejar la aplicación. también se have una documentación técnica y de gestión del proyecto.
### Explotación
Se prepara el software para distribuirlo y desplegarlo en los ordenadores de producción
### Mantenimiento
Se arreglan errores que sucedan cuando el programa se ejecuta con usuarios finales, y se realizan las ampliaciones necesarias.
