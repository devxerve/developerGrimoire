# :dev_javascript_original: Programación Web
# Front-end
Es la parte del software que interactúa con los usuarios. Es la parte visible que recolecta los datos de entrada de los usuarios. Está en la [[Programación Web (cliente)|parte del cliente]] en la arquitectura cliente-servidor.
# Back-end
Es la parte del software de la aplicación web que procesa las entradas desde el Front-end. Se encarga también de la conexión con la base de datos, procesamiento de la lógica de negocio y devolver la respuesta a la solicitud. Es la parte servidor en la arquitectura cliente-servidor.
