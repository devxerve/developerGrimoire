# Descarga E Instalación Del Programa
Entramos en el siguiente enlace y descargamos el instalador[https://developer.android.com/studio?hl=es-419](https://developer.android.com/studio?hl=es-419 "https://developer.android.com/studio?hl=es-419")
# Configuración Del SDK
Una vez instalado configuramos el SDK en la pestaña SDK Manager con la ultima versión disponible (16.0) y la versión 12 como mínimo.
![[Configuracion_sdk.png|Configuracion SDK]]

Tambien en la pestaña SDK tools instalamos el USB manager
![[Android_SDK_tools.png|Configuración SDK TOOLS]]

# Creación De Los Avd
En la pestaña principal vamos al desplegable more options y hacemos clic en Virtual Device manager
![[homeScreen.png|Selector AVD manager]]
Le damos a Create new Device

![[avd_manager.png|Pagina ppal AVD manager]]

## Móvil
Seleccionamos medium phone
![[selector_dispositivo.png|Selector dispositivo]]

Configuramos la versión y el nombre 

![[configuracion_avd.png|Configuracion AVD]]

## Tablet
Como ya tenemos un AVD creado la opción pasa a estar aquí en el AVD manager
![[nuevo_manager.png|Nuevo Manager]]

Seleccionamos medium tablet
![[selector_dispositivo_AVD(tablet).png|Selección medium Tablet]]

Configuramos la versión y el nombre 

![[configuracion_AVD_tablet.png|Selección opciones tablet|459x357]]
